# 🏗️ SQL Journey — Iron & Steel Trading Database

## About This Project
Learning SQL from scratch using a real-world Iron & Steel trading company database — **Guruji Kirpa Steel Trading**.

## 🗄️ Database Overview
| Table | Rows | Description |
|-------|------|-------------|
| suppliers | 50 | Raw material suppliers across North India |
| customers | 200 | Contractors, Builders, Fabricators |
| products | 30 | TMT Sariya, MS Angle, Channel, Plate, GI Pipe |
| purchases | 3,000 | 2 years purchase transactions |
| sales | 13,000 | 2 years sales transactions |

## 📚 Day-wise Progress
| Day | Topics Covered |
|-----|---------------|
| Day 01 | SELECT, INSERT, UPDATE, DELETE, WHERE |
| Day 02 | ORDER BY, COUNT, SUM, AVG, GROUP BY |
| Day 03 | Real Project Setup, CSV Import, JOIN, EXTRACT |

## 🛠️ Tools Used
- PostgreSQL 17
- pgAdmin 4
